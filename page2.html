Différents types de sélecteurs

Les différents types de sélecteurs
Valeur Description
p {}
 Sélecteur des éléments de type p
* {}
 Sélecteur générique : applique la modification à l'ensemble des éléments CSS
#id {}
 Sélecteur pour un id
.class {}
 Sélecteur pour toutes les classes

Combinaisons de sélecteurs

Combinaisons
Valeur Description
ul li {}
 Descendant Combinator : Sélectionne les élements "li" enfants des éléments "ul".
ul.class {}
 Chaining : Sélectionne les élements "ul" ayant une classe "class"
ul, ol {}
 Multiple selector : Sélectionne à la fois les éléments "ul" et les "ol".
Sélecteur d'attributs
Valeur Description
[href] {}
 Sélectionne tous les élements contrenant un attribut [href].
img[src*='winter'] {}
 Sélectionne les balises images contenant un attribut src avec le mot 'winter'

Lien vers la liste des pseudo-classes

Pseudo-class
Valeur Description
div:hover {}
 S'applique au moment où on survolle le div avec la souris.
div:active {}
 S'applique quand l'élément est actif : quand on clique dessus par exemple.
div:focus {}
 S'applique quand on focalise sur l'élément.
